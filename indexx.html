Protocol Overview

How the Protocol Works

CrossyFluffle runs entirely on-chain. Every game, score, and reward is verifiable on the Sepolia blockchain.

⬡

On-Chain Entry

Pay 0.001 ETH entry fee directly to the smart contract. The contract holds funds trustlessly — no escrow, no middlemen.

EVM Native
🔐

Signed Scores

Your score is cryptographically signed by our trusted backend wallet using ECDSA signatures — preventing any score manipulation.

ECDSA Verified

Auto-Distribute

Developer Tools

Deploy Your Own Instance

All contracts are open-source. Fork and deploy your CrossyFluffle instance in minutes.

deploy.js · Hardhat 
// ══ Rialo CrossyFluffle — Deploy ══
const { ethers } = require("hardhat");
async function main() {
  const [deployer] = await ethers.getSigners();
  
  const TRUSTED_SIGNER = process.env.TRUSTED_SIGNER;
  const BASE_URI = process.env.BASE_URI || "";
  
  const CrossyFluffle = await ethers
    .getContractFactory("CrossyFluffle");
    
  const contract = await CrossyFluffle
    .deploy(TRUSTED_SIGNER, BASE_URI);
    
  await contract.waitForDeployment();
  console.log("✓ Deployed:", await contract.getAddress());
}
main().catch(console.error);
